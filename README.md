# Smart Energy Monitoring Mobile Application

A Flutter-based smart energy monitoring mobile application for real-time visualization and analysis of electrical appliance behavior using experimentally observed datasets and machine learning-assisted monitoring.

---

## Features

* Real-time monitoring dashboard
* Voltage, Current, Power & Power Factor visualization
* Appliance-wise live graph visualization
* ML-assisted appliance health monitoring
* Dynamic animated UI
* Relay status indication
* Real observed dataset integration
* Lightweight optimized APK build

---

## Appliances Monitored

* LED Bulb
* Incandescent Bulb
* Table Fan
* Heater

---

## Parameters Visualized

* RMS Voltage (Vrms)
* RMS Current (Irms)
* Real Power
* Power Factor (PF)

## Project Objective

The objective of this project is to develop a smart electrical energy monitoring and appliance behavior visualization system capable of:

* Monitoring electrical parameters
* Visualizing appliance behavior
* Demonstrating ML-assisted appliance condition analysis
* Providing an intuitive mobile dashboard interface

---

## Note

This application is currently a demonstration-based mobile application developed using experimentally observed electrical datasets collected from real appliance testing.

The present version does not perform live ESP32-to-mobile real-time integration. Instead, it visualizes previously observed and recorded experimental data for demonstration, analysis, and presentation purposes.

However, the architecture and UI design of the application are fully extendable and can be integrated with real-time ESP32 + sensor-based monitoring systems in future implementations.
